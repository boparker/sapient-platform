## Environment Variables to Set

**Set these in the Railway web service (not the PostgreSQL service):**

### Core App Configuration

```
NODE_ENV=production
```

### Database (Auto-generated by Railway)
```
DATABASE_URL=${{Postgres.DATABASE_URL}}
```
*Railway auto-injects this when you reference the Postgres service*

### NextAuth Configuration

```
NEXTAUTH_SECRET=<GENERATE_THIS>
NEXTAUTH_URL=https://sapient-platform-production.up.railway.app
```

**How to generate NEXTAUTH_SECRET:**
```bash
openssl rand -base64 32
```

### Public Base URL

```
NEXT_PUBLIC_BASE_URL=https://sapient-platform-production.up.railway.app
```
*Update this to the actual Railway-generated domain after first deploy*

### Email Configuration (SendGrid)

```
SMTP_HOST=smtp.sendgrid.net
SMTP_PORT=587
SMTP_USER=apikey
SMTP_PASS=<SENDGRID_API_KEY>
FROM_EMAIL=noreply@andreabertoli.com
```

**SendGrid setup needed:**
1. Create SendGrid account (or use existing)
2. Generate API key with "Mail Send" permissions
3. Verify sender email: noreply@andreabertoli.com (or similar)
